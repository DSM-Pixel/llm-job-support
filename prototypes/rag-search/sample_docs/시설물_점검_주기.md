# 도로 시설물 점검 주기 (샘플 문서)

> ⚠️ 본 문서는 데모용 MOCK 자료입니다. 실제 규정이 아닙니다.

## 시설물별 정기 점검 주기
- **가드레일(방호울타리)**: 연 1회. 충돌 흔적·부식·체결 상태 확인.
- **도로 표지판**: 연 2회. 야간 반사 성능, 기둥 기울기 점검.
- **도로 조명(가로등)**: 분기 1회. 점등 상태, 누전 여부 확인.
- **배수 시설(측구·집수정)**: 우기 전 1회 집중 점검. 토사 퇴적 제거.
- **중앙분리대**: 연 1회. 파손·이격 점검.

## 점검 결과 처리
1. 점검 항목별로 양호/주의/불량 3단계로 판정한다.
2. '불량'은 30일 이내 보수 계획을 수립한다.
3. 안전에 직접 영향을 주는 '불량'(예: 가드레일 탈락)은 즉시 조치한다.

## 기록 관리
점검 일자, 점검자, 위치(GPS), 사진, 판정 결과를 점검 대장에 기록하고 5년간 보관한다.
